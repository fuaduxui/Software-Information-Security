# Software Information Security (CS-922di.e)

Laboratory works for the academic course. All programs are written in pure Python without external dependencies.

## Content:
* **lab1_2_security.py** — User authorization system with SHA-256 password hashing.
* **lab3_security.py** — Installer script with hardware signature binding.
* **lab4_data_compression.py** — Data archiver using RLE compression algorithm.
* **lab5_encryption.py** — Symmetric block cipher logic.
* **lab6_streaming_cipher.py** — Stream data protection using RC4 algorithm.
* **lab7_digital_signature.py** — Asymmetric digital signature with RSA and MD2.
* **lab8_steganography.py** — Steganographic data hiding inside BMP files via LSB method.
