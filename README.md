# Software Information Security 
*Fuad Mammadov (KH-922 di.e)*

Laboratory works for the academic course. All programs are written in pure Python.

## Content:
* **Lab1-2.py** — User authorization system with SHA-256 password hashing.
* **Lab3.py** — Installer script with hardware signature binding.
* **Lab4.py** — Data archiver using RLE compression algorithm.
* **Lab5.py** — Symmetric block cipher logic.
* **Lab6.py** — Stream data protection using RC4 algorithm.
* **Lab7.py** — Asymmetric digital signature with RSA and MD2.
* **Lab8.py** — Steganographic data hiding inside BMP files via LSB method.
